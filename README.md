# Course API of Udemy - Javascript e Typescript do básico ao avançado 2021

### In a module of this course, we made an API of a school, with users and students login, using tokens.

## Technologies and libs used:

* NodeJS
* Express
* Dotenv
* Multer
* JWT
* Bcrypt
* Sequelize
* Google cloud
* MariaDB
